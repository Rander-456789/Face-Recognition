# Face Recognition Pipeline: End-to-End System

Репозиторий содержит комплексное решение задачи распознавания лиц (Face Recognition Pipeline) на базе датасета CelebA. В проекте представлены основные этапы работы над пайплайном распознавания лиц, такие как: создание датасета, выравнивание датасета, обучение модели для выделения ключевых точек, обучение двух моделей для решения задачи распознавания лиц на двух разных лоссах (CE loss и ArcFace). Также решены дополнительные задачи, такие как реализация метрики Identification Rate Metric, а также составление отчета по существующим в сообществе открытым библиотекам для решения данной задачи.

---

## Общая структура проекта

Для удобства проверки проект разделен на изолированные модули. Ниже представлена карта соответствия файлов заданиям курса:

| Тип задания | Описание задачи | Файл в репозитории |
| :--- | :--- | :--- |
| **Данные** | Подготовка датасета, фильтрация CelebA и расчет статистики по классам. | `Dataset_creation.ipynb` |
| **Основное** | **Выравнивание:** Stacked Hourglass Network для поиска ключевых точек лица и Аффиновское выравнивание. | `Stage_1.ipynb` |
| **Основное** | **Метрическое обучение:** Обучение моделей на CE loss и ArcFace. | `Stage_2.ipynb` |
| **Основное** | **Сборка пайплайна:** Интеграция детектора, выравнивания и распознавания. Демонстрация расстояний. | `Stage_3.ipynb` |
| **Дополнительное** | **Метрики FR:** Реализация и расчет метрики Identification Rate Metric. | `Dop_ques_1.ipynb` |
| **Дополнительное** | **Аналитический отчет:** Ресерч и сравнение 3 open-source библиотек для распознавания лиц. | `Dop_ques_4.ipynb` |
| **Данные** | Таблица разделения выборки, соответствия ID и меток классов. | `data/split_info.csv` |
